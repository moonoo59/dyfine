## 🔴 수정 완료된 버그 (Resolved)

| # | 영역 | 증상 | 원인 | 해결 방법 |
|---|------|------|------|----------|
| B-01 | 계좌 관리 | 계좌 추가 후 리스트에 표시되지 않음 | `useAccounts.ts`에서 존재하지 않는 `account_group` 컬럼으로 정렬 → 쿼리 실패 | 해당 정렬 조건 제거 |
| B-02 | 예산 관리 | 예산 저장이 안됨 | 코드가 `budget_templates.amount` 사용, 실제 DB는 `budget_template_lines.monthly_amount` | 올바른 테이블/컬럼으로 전환 |
| B-03 | 대시보드 | 통화 기호가 숫자 뒤에 표시, 지출에 마이너스(-) 부호 | UI 포맷팅 오류 | `₩` 기호를 앞으로 이동, ±기호 제거 |
| B-04 | 온보딩 | 가구 생성 시 RLS 정책 오류 | `households` INSERT 정책 미적용 | `create_household_with_owner` RPC 함수로 우회 |
