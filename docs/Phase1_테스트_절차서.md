## 테스트 1: 계좌 관리 (`/accounts`)

| # | 테스트 항목 | 기대 결과 | 결과 |
|---|-----------|---------|------|
| 1-1 | 좌측 메뉴에서 **"계좌 관리"** 클릭 | 계좌 관리 페이지로 이동 | PC에서는 상단에 메뉴가 표시|
| 1-2 | **"새 계좌 추가"** 버튼 클릭 → 이름: `신한은행 입출금`, 유형: `checking`, 초기잔액: `1000000` 입력 후 저장 | 목록에 계좌가 추가됨 |❌ 대시보드에 금액 반영은 되지만, 새 계좌가 리스트업 되지 않음 |
| 1-3 | 계좌를 2~3개 더 추가해 보기 (예: `국민카드`, `토스뱅크 적금`) | 모두 정상 추가됨 |❌ 대시보드에 금액 반영은 되지만, 새 계좌가 리스트업 되지 않음 |
추가 코멘트
 1) 계좌명, 은행/증권사, 계좌번호, 예금주도 입력하여 관리 할 것
 2) 메뉴 클릭 시 "데이터를 불러오는 중" 표시가 3초 정도 지속됨
 3) 외부/신용카드의 기능을 모르겠음
 4) 계좌 유형을 상세하게 나눌 것

## 테스트 2: 분류(카테고리) 관리 (`/categories`)

| # | 테스트 항목 | 기대 결과 | 결과 |
|---|-----------|---------|------|
| 2-1 | **"분류 관리"** 메뉴 클릭 | 카테고리 관리 페이지로 이동 | 동작 확인|
| 2-2 | 대분류(L1) 추가: `식비` | 목록에 "식비" 표시됨 | 동작 확인|
| 2-3 | 소분류(L2) 추가: 부모를 `식비`로 선택 → `점심` | "식비" 아래에 "점심"이 하위로 표시됨 | 동작 확인|
| 2-4 | 추가 카테고리 생성: `교통비`, `월급(수입)` 등 | 모두 정상 추가됨 | 동작 확인|
추가 코멘트
 1) 거래 내역 추가시 카테고리 반영은 되지만, L1, L2로 나누어지는 것이 좋을 듯. L1을 선택하면 그에 맞는 하위 항목이 나오게
 2) 카테고리 추가시, L1은 전체 카테고리 추가 그 하위항목은 각 대분류(L1)에서 추가 버튼 같은것으로 추가하면 더 직관적일 듯
 3) 태그 관리 : 전기요금, 가스요금, 배달식비, 식료품비 등의 상세 항목은 태그로 관리하면 좋을 듯
 4) 계좌관리, 분류관리 기능은 "설정" 메뉴가 있고 그 아래에서 관리하는 걸로
 5) 지출/수입 별 카테고리 별도 설정

## 테스트 3: 거래 내역 - 지출 입력 (`/transactions`)

| # | 테스트 항목 | 기대 결과 | 결과 |
|---|-----------|---------|------|
| 3-1 | **"거래 내역"** 메뉴 클릭 | 거래 내역 페이지로 이동, "등록된 거래 내역이 없습니다" 표시 | 확인 |
| 3-2 | **"빠른 추가"** 버튼 클릭 | 모달(팝업) 창이 열림 | 확인 |
| 3-3 | **지출** 탭 선택 → 날짜: 오늘, 금액: `12000`, 출금 계좌: `신한은행 입출금`, 카테고리: `식비`, 메모: `점심 회식` → 저장 | 모달 닫히고 거래 목록에 새 항목 표시 (빨간 `-12,000`) | 계좌등록이 안되어 선택 불가 |
| 3-4 | 목록에서 방금 추가한 거래의 **유형 태그**가 `EXPENSE`인지 확인 | 빨간색 EXPENSE 뱃지 표시 | 계좌 등록이 안되어 저장 불가하여 확인 안됨 |
추가 코멘트
 1) 금액입력시, 1,000원 단위로 쉼표

## 테스트 7: 예산 관리 (`/budgets`)

| # | 테스트 항목 | 기대 결과 | 결과 |
|---|-----------|---------|------|
| 7-1 | **"예산 관리"** 메뉴 클릭 | 예산 관리 페이지로 이동 | 확인 |
| 7-2 | **"새 예산 설정"** 클릭 → 카테고리: `식비`, 금액: `300000` → 저장 | 카드 형태로 예산 표시됨 | 저장 안됨 |
| 7-3 | 프로그레스 바(진행률) 확인 | 아까 입력한 12,000원 지출이 반영되어 바가 4% 정도 채워짐 | |
| 7-4 | **"수정"** 버튼 클릭 → 금액을 `10000`으로 변경 → 저장 | 프로그레스 바가 빨갛게 변하며 "예산을 초과했습니다!" 메시지 표시 | |
추가 코멘트
 1) 예산 관리 -> 예산 이름 변경
 2) 월별 예산 설정 필요
 3) 월별 항목, 금액 같을 시 "전월 불러오기" 기능
